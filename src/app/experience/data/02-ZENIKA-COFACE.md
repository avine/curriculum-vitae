### Architecte logiciel Front-end – Refonte SI Assurance

Coface, leader mondial de l'assurance-crédit, engage une transformation digitale majeure de son SI legacy pour accompagner sa croissance internationale et moderniser l'expérience utilisateur.

#### Réalisations & impact

- Conception et développement from scratch : librairie Angular (Design System) + starter kit dans Monorepo Nx (pour les applications métier)
- Application de démo interactive de la librarie + documentation technique
- Intégration PrimeNG alignée avec la charte graphique + documentation interactive
- Intégration dans une application pilote + Transfert de connaissance

**Stack:** Angular 16-17, NgRx, Ngx-Translate, PrimeNG, Nx
